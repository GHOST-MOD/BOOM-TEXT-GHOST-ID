User Text Auto Typing App with Line Limit

About

This application allows users to type their own text and set a line limit for auto-typing. The auto-typing feature will repeat the user's text up to the specified line limit, providing a dynamic and engaging experience.

The background of the application features a vibrant neon gradient animation, creating a visually appealing interface. The text areas and buttons are styled with Tailwind CSS to ensure a modern and responsive design.
